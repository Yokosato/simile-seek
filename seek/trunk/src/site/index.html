Seek

“Seek” adds faceted browsing features to Mozilla Thunderbird and lets you search through your email more effectively.

How to Install
  1. Right-click the download link (left) and choose “Save Link As...” to download and save the file to your hard disk.
  2. In Mozilla Thunderbird, open “Add-ons” from the “Tools” menu.
  3. Click the “Install” button, and locate/select the file you downloaded and click “OK”.
How to Use

Seek's panel can be toggled in the Tools menu. There is also a keyboard shortcut: Ctrl-K (Windows) or Command-K (Mac).
